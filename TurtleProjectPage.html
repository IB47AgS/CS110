About the Project

This project is about creating a graphical scene using Python Turtle module. I used the provided functions like "draw_rectangle", "draw_square", "draw_triangle", "draw_circle", "draw_polygon", "draw_curve", etc. to declare functions that draw different shapes.

In this improved version of my Project 4, I made the code better by: breaking up the large main function into smaller functions, makes the code easier to read and change. Creating reusable functions for things that appear multiple times, like clouds and trees. Using what we learned in class about making code more organized. This refactoring did not add new features but made the codebase more organized and easier to understand.

My Code

This function creates a cloud shape using multiple overlapping circles. The parameters allow for flexible positioning and scaling:

• t: The turtle is the graphics object for drawing
• x, y: Starting position coordinates for the pattern
• size: Scale factor (default 1.0) to adjust larger or smaller

import turtle
def jump_to(t, x, y):
  t.penup()
  t.goto(x, y)
  t.pendown()
def draw_triangle(t, x, y, size):
  jump_to(t, x, y)
  t.begin_fill()
  for _ in range(3):
    t.forward(size)
    t.right(120)
  t.end_fill()
def draw_cloud(t, x, y):
  for i in range(4):
    jump_to(t, x + i * 30, y)
    t.begin_fill()
    t.circle(25)
    t.end_fill()
def draw_scene(t):
  draw_triangle(t, 50, -20, 60)
  draw_cloud(t, 0, 180)
def main():
  screen = turtle.Screen()
  t = turtle.Turtle()
  t.speed(0)
  draw_scene(t)
  screen.mainloop()
if __name__ == '__main__':
  main()
